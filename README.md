# Portfolio - Mobile Application

Aplicativo mobile desenvolvido em React Native com Expo, reunindo portfólio pessoal, experiências acadêmicas, projetos desenvolvidos e um mini game interativo integrado diretamente à aplicação.

O projeto foi desenvolvido como atividade acadêmica, utilizando conceitos modernos de desenvolvimento mobile, interface responsiva, persistência local de dados e mecânicas interativas em tempo real.

# Visão Geral do Projeto

Este projeto tem como objetivo transformar um currículo tradicional em uma experiência mobile moderna, dinâmica e interativa.

A aplicação apresenta informações pessoais, experiências acadêmicas, projetos desenvolvidos e uma área de entretenimento integrada através de um mini game arcade desenvolvido inteiramente em React Native.

O sistema foi projetado buscando unir:

* Design moderno
* Navegação fluida
* Interatividade
* Responsividade
* Organização de conteúdo
* Performance mobile
* Experiência do usuário
